Todo App

Todo App is a simple web application for managing a task list, built with Django. The app allows users to create new tasks, mark them as completed, and delete unnecessary tasks.

This project is designed to demonstrate the basic functionality of web applications and provides a convenient interface for task management.

Project Features:

1) Create and delete tasks

2) Mark tasks as completed

3) Minimalist and user-friendly interface

4) Easily extendable code for further development

This project is ideal for beginners who want to learn the basics of Django and practice working with databases, templates, and forms.
